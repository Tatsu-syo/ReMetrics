2.1 Run program

Run ReMetrics.exe.

2.2 Operation

Basic operation

When window displayed, you see window element name and size.
Please input size in textbox or change size clicking spin button.

When setting ended, press OK button.
You can change wildow element size.(Remark, OS adjust element size)

When you press cancel button, program exits without action.

Menu

  • File
    • Load window settings...
      Load window settings from file. Please select window settings file.
      You can create window settings file by Save window settings...
    • Save window settings...
      Save window settings into file.
      Please input window settings file name.
    • Set window settings and quit
      Set window settings and quit program. Same as OK button.
    • Exit
      Exit program without any action. Same as Cancel button.
  • Preset
    You can choose window settings each Windows.
    • Windows 2000
    • Windows XP Classic
    • Windows XP
    • Windows Vista
    • Windows 7 Basic
    • Windows 7
    • Windows 8/8.1
    • Windows 10
  • Tols
    • Set window settings in separated thread.
      If some resident programs block program exit, check this option.
  • Help
    • Help
      Show help file.
    • Windows Version
      Show Windows version information.
      Use this information in troubleshooting.
    • About Re-Metrics
      Show this program's version.

2.3 Setting items

  • Border width
    Window's border width.
    On Windows 10, you can adjust window resize boundary.
  • Title bar width
    Button width in title bar.
  • Title bar height
    Height of title bar.
  • Scroll bar width
    Width of scroll bar and it's button.
  • Scroll bar height
    Height of scroll bar and it's button.
  • Palette width
    Width of palette and it's button.
  • Palette height
    Height of palette and it's button.
  • Menu width
    Menu bar's button width in MDI application.
  • Menu height
    Height of menu bar and Menu bar's button.
  • Padded border width
    Width of window padding.
  • Icon horizontal spacing
    Width of icon spacing.
  • Icon vertical spacing
    Height of icon spacing.

Icon horizontal spacing and icon vertical spacing are enabled after "Auto arrange icons" or "Align icons to grid".

2.4 Command line option

You can set window settings in command line
Re-Metrics window settings file name [-set]

You can create window settings file by this program.
If window settings file countains white space, surround file name in double quote.

If you specify -set after window setting file, program sets window settings from window settings file and program exits.
Use this option in batch file or launcher.

Example 1. Load MySet.ini on running.

ReMetrics MySet.ini

Example 2. Setting window settings from Your Settings.ini

ReMetrics "Your Settings.ini" -set
