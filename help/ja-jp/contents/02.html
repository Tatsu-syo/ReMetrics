2.本ソフトの使い方

2.1 始めに

　エクスプローラーから左のアイコンの実行ファイルReMetrics.exeをダブルクリックするなどして、ReMetrics.exeを実行してください。
　下記の画面が表示されます。


2.2 操作方法

基本操作

　画面が表示されると画面の各部分の名称と設定されているサイズが表示されます。
　テキストボックスにサイズを入力するか、横のスピンボタンをクリックして画面各部の幅・高さを入力してください。

　設定が終わったらOKボタンを押下すると画面の各部分の幅・高さを指定した数値に設定します。
　実際の画面の各部分の幅・高さは入力した数値に対し、OSの側で調整されたサイズになります。

　キャンセルボタンを押下すると何もせずに本ソフトを終了します。

メニュー

　メニューバーの各メニューの機能を説明します。

  • ファイル
    • ウインドウ設定を読み込む(O)...
      画面の各部分の幅・高さを保存したファイルを読み込みます。
      「ウインドウ設定を保存する(A)...」メニューで画面の各部分の幅・高さを保存したウインドウ設定ファイルを選択してください。
    • ウインドウ設定を保存する(A)...
      画面の各部分の幅・高さを保存したします。
      ダイアログで保存するウインドウ設定ファイルの名前を入力してください。
    • 各項目を設定して終了(S)
      画面に入力された各項目の内容に沿って画面の各部分の幅・高さを設定します。
    • 終了(X)
      キャンセルボタンと同様に何もせずに本ソフトを終了します。
  • プリセット(P)
    　以下に示す各OSの画面の各部分の幅・高さを設定項目に設定します。
    • Windows 2000
    • Windows XP Classic
    • Windows XP
    • Windows Vista
    • Windows 7 ベーシック
    • Windows 7
    • Windows 8/8.1
    • Windows 10
    • Windows 11
  • ツール(T)
    • 別スレッドで幅・高さを変更する。(A)
      環境依存のトラブルシューティング用の設定を行います。
      SetPoint(Logicoolのマウス・キーボードユーティリティ)等を起動した環境で設定を行った時に応答なしになる場合、本メニューをチェックしてから設定を行ってください。
  • ヘルプ(H)
    • ヘルプの表示(C)
      ヘルプファイルを表示します。
    • Windowsのバージョン(W)
      Windowsのバージョン情報を表示します。
      Windowsのバージョンを本ソフトが正しく認識しているかどうかの確認や動作報告にお使いください。
    • バージョン情報(A)
      本ソフトのバージョン情報を表示します。

2.3 設定項目の説明

　画面上の各項目の内容は次のとおりです。

  • ウインドウの境界
    　ウインドウ枠の太さです。
    　Windows 10で大きい値を指定するとウインドウ枠の見た目は変わりませんが、ウインドウのサイズを変更する際、ウインドウの縁から離れていてもリサイズができるようになります。
  • タイトルバーの幅
    　タイトルバーのボタンの幅です。(Windows 2000/XP Windowsクラシック/Vista Windowsクラシック/7 Windowsクラシックで有効)
  • タイトルバーの高さ
    　タイトルバーの高さです。
  • スクロールバーの幅
    　スクロールバーとスクロールバーのボタンの幅です。
  • スクロールバーの高さ
    　スクロールバーとスクロールバーのボタンの高さです。
  • パレットの幅
    　パレットのボタンの幅です。
  • パレットの高さ
    　パレットとパレットのボタンの高さです。
  • メニューの幅
    　MDIアプリケーションのメニューバーに表示されるボタンの幅です。
  • メニューの高さ
    　メニューバーとメニューバーに表示されるボタンの高さです。
  • ウインドウの境界の間隔
    　ウインドウ枠とウインドウの中身(クライアント領域)の間の幅です。
    　Windows 10で大きい値を指定するとウインドウ枠の見た目は変わりませんが、ウインドウのサイズを変更する際、ウインドウの縁から離れていてもリサイズができるようになります。
  • アイコンの間隔(横)
    　「アイコンを等間隔に整列する」を指定したときや「アイコンの自動整列」を指定したときのデスクトップアイコンの横の間隔です。
  • アイコンの間隔(縦)
    　「アイコンを等間隔に整列する」を指定したときや「アイコンの自動整列」を指定したときのデスクトップアイコンの縦の間隔です。

　「アイコンの間隔(横)」「アイコンの間隔(縦)」はデスクトップを右クリックして「表示」の「アイコンを等間隔に整列する」もしくは「アイコンの自動整列」のチェックをいったん外した後再度チェックすることで有効になります。

2.4 コマンドラインオプション

　コマンドラインからオプションを指定することで起動時にウインドウ設定を読み込むことができるほか、指定したウインドウ設定で自動的にウインドウ各部の幅・高さを設定することができます。

　コマンドラインの書式は以下の通りです。
ReMetrics ウインドウ設定ファイル名 [-set]

　ウインドウ設定ファイルには本プログラムでウインドウ設定を保存したiniファイルを指定します。
　ウインドウ設定ファイル名に空白を含む場合はファイル名を"(ダブルクォーテーション)で囲ってください。

　ウインドウ設定ファイル名の後に空白を入れて-setと入力すると指定したウインドウ設定ファイルの内容でウインドウ各部の幅・高さを設定してプログラムを終了します。
バッチファイルなどで使用するとあらかじめ用意したウインドウ設定ファイルの内容にウインドウ各部の幅・高さを一発で設定することができて便利です。

例1. MySet.iniというウインドウ設定ファイルを起動時に読み込む場合

　MySet.iniというウインドウ設定ファイルを起動時に読み込む場合はコマンドプロンプトなどから以下のように入力します。
ReMetrics MySet.ini

例2. Your Settings.iniというウインドウ設定ファイルの内容にウインドウ各部の幅・高さを設定する場合

　Your Settings.iniというウインドウ設定ファイルの内容にウインドウ各部の幅・高さを設定する場合はコマンドプロンプトなどから以下のように入力します。
ReMetrics "Your Settings.ini" -set
