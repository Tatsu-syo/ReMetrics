1.Getting started

1.1 このプログラムについて

このプログラムはWindows 8.1/10/11でタイトルバーの高さなどウインドウ幅の設定を行うツールです。
Windows 8.1でなぜかタイトルバーの高さなどウインドウ幅の設定を行う機能がなくなったので作成しました。

1.2 Target OS

このプログラムはWindows 8.1/10/11で動作します。
Windows Vsita/7でも動作しますが、意味はありません(もともと機能がある)。

1.3 構成ファイル
